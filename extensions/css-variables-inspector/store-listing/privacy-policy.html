CSS Variables Inspector - Privacy Policy

Last updated: March 23, 2026

Overview

CSS Variables Inspector is a Chrome DevTools extension that displays CSS custom properties (variables) for inspected elements in a sidebar panel. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

CSS Variables Inspector does not collect, store, transmit, or share any data. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No page content or CSS data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.

How It Works

CSS Variables Inspector uses the chrome.devtools.inspectedWindow API to read CSS custom properties from the currently selected element in the Elements panel. All data is processed locally in memory and is discarded when the DevTools panel is closed. No data is persisted to disk or transmitted externally.

Permissions

The extension requests the following permissions:

  • storage — Used for saving user preferences (such as grouping mode). No page data is stored.

Third-Party Services

CSS Variables Inspector does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero data collection.

Contact

If you have questions about this privacy policy, you can reach out through the extension's Chrome Web Store listing.
