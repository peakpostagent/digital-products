Privacy Policy

Last updated: April 24, 2026 · v0.1.0 scaffold draft

Etsy SEO Tag Generator is a Chrome extension that scans live Etsy listing pages and suggests improved tags. This privacy policy will be finalized before the public CWS launch.

The short version (v0.1.0 scaffold): This extension reads listing data from Etsy pages you actively view. Nothing is sent to external servers in v0.1.0. When the AI-suggestion feature ships, listing data will be transmitted to our backend for processing.

What Data We Access

When you click the extension's icon while on an etsy.com/listing/* page, the extension reads the following from the live page:

  • Listing title, price, description, tags, category breadcrumbs, and image count
  • The current page URL

This data is read ONLY when you actively open the extension popup. It is not collected silently in the background.

What Data We Collect (v0.1.0)

None. The current scaffold does not transmit any data to external servers. All listing data stays in your browser.

Third-Party Services (planned for v1.0.0+)

When the AI tag-suggestion feature ships, listing data (title, description, current tags, breadcrumbs) will be transmitted to:

  • Our hosted backend on Vercel for processing
  • An LLM provider (likely OpenAI gpt-4o-mini) for tag-suggestion generation

This privacy policy will be updated to fully disclose these data flows before the AI feature is enabled.

Permissions Explained

  • storage — Reserved for caching scan results and user preferences. Not currently used in v0.1.0.
  • activeTab — Allows the popup to send a message to the active Etsy tab to extract listing data on demand.
  • Host permission (https://*.etsy.com/*) — Required so the content script can read listing data from Etsy pages. The extension only activates on listing URLs.

Children's Privacy

This extension is not directed at children under 13.
