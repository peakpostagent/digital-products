Meta Tag Viewer - Privacy Policy

Last updated: April 6, 2026

Overview

Meta Tag Viewer is a Chrome extension that lets developers inspect SEO meta tags on any web page with one click. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

Meta Tag Viewer does not collect, store, transmit, or share any data externally. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No meta tag data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.

How It Works

Meta Tag Viewer uses the chrome.scripting API to execute a script in the current tab's page context. This script reads the page's <title>, <meta>, and <link> elements to gather meta tag information. All data is processed locally and displayed in the extension popup. Only user preferences (such as section collapse state) are stored in chrome.storage.local on your device.

Permissions

The extension requests the following permissions:

  • storage — Used for storing user preferences locally on your device.
  • activeTab — Required for the popup to identify the currently active tab and read its meta tags.
  • scripting — Used to execute a read-only script in the page context to extract meta tag values from the DOM.

Third-Party Services

Meta Tag Viewer does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero external data collection.

Contact

If you have questions about this privacy policy, you can reach out through the extension's Chrome Web Store listing.
