Meta Tag Viewer - Privacy Policy

Last updated: May 12, 2026

Overview

Meta Tag Viewer is a Chrome extension that lets developers inspect SEO meta tags on any web page with one click. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

Meta Tag Viewer does not collect, store, transmit, or share any data externally. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No meta tag data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.
  • Nothing is persisted between popup sessions — all state lives only while the popup is open.

How It Works

Meta Tag Viewer uses the chrome.scripting API to execute a single read-only function in the current tab's page context. This function reads the page's <title>, <meta>, and <link rel="canonical"|"icon"> elements to gather meta tag information. All data is processed locally and displayed in the extension popup. When the popup closes, the data is discarded.

The optional "Copy to clipboard" feature writes a text summary of the current tag inspection to your system clipboard via the standard navigator.clipboard.writeText() API — only when you click the Copy button.

Permissions

The extension requests the following permissions:

  • activeTab — Required so the popup can identify the user's active tab and read its meta tags on demand. Scoped to a single tab the user explicitly opened the popup on.
  • scripting — Required to execute a single static read-only function (extractMetaTags, defined in popup/popup.js) in the active tab's page context. No remote code, no dynamic code, no eval — the injected function is hard-coded in the extension source and only reads DOM elements; it does not modify them.

What the extension does NOT do

  • No storage permission requested — nothing is saved between sessions.
  • No host permissions — the extension only runs when you explicitly click its icon on the currently-active tab.
  • No background scripts or service workers that run when the browser is idle.
  • No clipboard read — the extension can write to your clipboard only when you click Copy, never read from it.

Third-Party Services

Meta Tag Viewer does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero external data collection.
