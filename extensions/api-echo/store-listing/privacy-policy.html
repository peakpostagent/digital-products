API Echo - Privacy Policy

Last updated: March 16, 2026

Overview

API Echo is a Chrome DevTools extension that displays network requests and responses in a focused panel. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

API Echo does not collect, store, transmit, or share any data. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No network request data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.

How It Works

API Echo uses the chrome.devtools.network API to observe network requests within the DevTools panel. All captured request and response data is held temporarily in memory and is discarded when the DevTools panel is closed or when the inspected page navigates away. No data is persisted to disk or transmitted externally.

Permissions

The extension requests the following permissions:

  • storage — Used for saving user preferences (such as filter settings). No request data is stored.
  • activeTab — Required for the DevTools panel to interact with the currently inspected tab.

Third-Party Services

API Echo does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero data collection.

Contact

If you have questions about this privacy policy, you can reach out through the extension's Chrome Web Store listing.
