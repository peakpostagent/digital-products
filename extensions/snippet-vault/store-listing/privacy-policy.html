Snippet Vault - Privacy Policy

Last updated: March 23, 2026

Overview

Snippet Vault is a Chrome extension that lets you save, tag, and search code snippets. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

Snippet Vault does not collect, store externally, transmit, or share any data. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No snippet data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.

How It Works

Snippet Vault stores all code snippets locally on your device using chrome.storage.local. Snippets are never transmitted externally. The right-click context menu feature uses the contextMenus permission to allow saving selected text as a snippet.

Permissions

The extension requests the following permissions:

  • storage — Used to save your code snippets and settings locally on your device.
  • contextMenus — Adds a right-click menu option to save selected text as a code snippet.

Third-Party Services

Snippet Vault does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero data collection.

Contact

If you have questions about this privacy policy, you can reach out through the extension's Chrome Web Store listing.
