Privacy Policy

Snippet Vault Chrome Extension — Last updated: March 2026

Summary

Snippet Vault does not collect, transmit, or share any user data. All information stays on your device.

Data Storage

All code snippets, tags, and settings are stored locally on your device using Chrome's built-in storage API (chrome.storage.local). No data is sent to any external server, database, or third-party service.

Data Collection

This extension does not collect any personal information, browsing history, analytics, telemetry, or usage data of any kind.

Third-Party Services

This extension does not use any third-party services, APIs, or tracking tools. There are no ads, no analytics, and no external network requests.

Permissions

The extension requests the following permissions:

  • storage — Used to save your code snippets locally on your device.
  • contextMenus — Used to add a right-click menu option for saving selected text as a snippet.

Data Export

You can export your snippets as a JSON file at any time using the Export feature. This file is saved to your local downloads folder. No data is uploaded.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date. The core commitment to local-only storage will not change.

Contact

If you have questions about this privacy policy, please open an issue on the extension's support page.
