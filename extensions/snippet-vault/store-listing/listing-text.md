# Chrome Web Store Listing - Snippet Vault

## Name
Snippet Vault

## Short Description (132 chars max)
Save, tag, and search code snippets locally. Copy any snippet with one click. No accounts needed — your code stays on your device.

## Detailed Description
Snippet Vault is a lightweight code snippet manager that lives in your browser toolbar. Save reusable code, tag it by language, and find it instantly with real-time search.

**Key Features:**
- Save code snippets with a title, language label, and custom tags
- Real-time search across titles, tags, languages, and code content
- One-click copy to clipboard
- Sort by newest, oldest, A-Z, or language
- Right-click any selected text on a web page to save it as a snippet
- Import and export your snippet library as JSON for backup or sharing
- Keyboard shortcut (Ctrl+Shift+S) to open instantly
- Badge counter shows your total snippet count

**Supported Languages:**
JavaScript, Python, HTML, CSS, SQL, Bash, Go, Rust, TypeScript, and more.

**Privacy First:**
All snippets are stored locally on your device using Chrome's built-in storage. No accounts, no cloud sync, no data collection. Your code never leaves your browser.

**Built for Developers:**
Dark theme with monospace code fonts. Color-coded language tags. Tab key works inside the code editor. Clean, distraction-free interface designed for quick access.
