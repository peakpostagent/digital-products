Privacy Policy — DevTools Decoder

Last updated: April 6, 2026

Summary

DevTools Decoder does not collect, transmit, or share any personal data. Everything stays on your device.

Data collection

DevTools Decoder collects no personal information. The extension stores the following data locally on your device using Chrome's built-in chrome.storage.local API:

  • Your last active tab preference (a single string value).

Data sharing

DevTools Decoder does not transmit any data to external servers, analytics services, or third parties. There are no network requests made by this extension.

Data processing

All encoding, decoding, hashing, and formatting operations are performed entirely within your browser. No input data is sent anywhere. Hash generation uses the browser's built-in Web Crypto API.

Data retention

Only the last active tab name is stored. No user input or output is persisted. You can clear all stored data at any time by uninstalling the extension or clearing its storage through Chrome's extension settings.

Permissions

DevTools Decoder requests the following Chrome permission:

  • storage — To remember which tool tab was last used, so it opens to the same tab next time.

Third-party services

DevTools Decoder uses no third-party services, SDKs, analytics, or tracking tools.

Changes to this policy

If this privacy policy is updated, the changes will be noted here with a revised "last updated" date.
