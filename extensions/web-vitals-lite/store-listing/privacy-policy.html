Web Vitals Lite - Privacy Policy

Last updated: March 23, 2026

Overview

Web Vitals Lite is a Chrome extension that displays Core Web Vitals (LCP, CLS, and INP) in a floating badge on any webpage. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

Web Vitals Lite does not collect, store, transmit, or share any data. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No performance data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.

How It Works

Web Vitals Lite uses the browser's built-in Performance API (PerformanceObserver) to measure Largest Contentful Paint, Cumulative Layout Shift, and Interaction to Next Paint. All metric calculations happen entirely within your browser. The floating badge and popup display data that exists only in memory and is discarded when the tab is closed or navigated away.

Permissions

The extension requests the following permissions:

  • storage — Used for saving user preferences (such as badge visibility). No performance data is stored.
  • activeTab — Required for the popup to retrieve vitals data from the currently active tab.

Third-Party Services

Web Vitals Lite does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero data collection.

Contact

If you have questions about this privacy policy, you can reach out through the extension's Chrome Web Store listing.
