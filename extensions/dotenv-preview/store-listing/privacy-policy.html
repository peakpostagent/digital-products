Privacy Policy

Last updated: March 11, 2026

DotEnv Preview is a Chrome extension that syntax-highlights and organizes .env files on GitHub, GitLab, and Bitbucket.

The short version: We don't collect, store, or transmit any of your data. Everything happens locally in your browser.

What Data We Access

DotEnv Preview accesses the following data solely within your browser:

  • Your display preferences: Settings such as theme, grouping, and masking preferences are stored locally using Chrome's built-in storage API (chrome.storage.local). They never leave your browser.
  • File content on code hosting pages: When you view a .env file on GitHub, GitLab, or Bitbucket, the extension reads the visible file content from the page to parse and format it. This data is processed locally and is not stored or transmitted.

What Data We Collect

None. DotEnv Preview does not collect, transmit, or store any personal data on external servers. Specifically:

  • No analytics or tracking scripts
  • No cookies
  • No user accounts or sign-ups
  • No network requests to external servers
  • No browsing history collection
  • No personally identifiable information (PII) is ever transmitted
  • No file content is stored beyond the current page view

Where Data Is Stored

Your display preferences (auto-format, masking, grouping, line numbers, theme) are stored exclusively in your browser's local storage via chrome.storage.local. This data:

  • Stays on your device
  • Is only accessible to the DotEnv Preview extension
  • Can be deleted at any time by clearing the extension's data or uninstalling it
  • Is not synced across devices

Third-Party Services

DotEnv Preview does not use any third-party services, APIs, analytics platforms, or advertising networks.

Permissions Explained

  • storage: Used to save your display preferences (theme, grouping, masking settings) locally.
  • activeTab: Used to read the currently visible file content on supported code hosting pages.
  • Host permissions (github.com, gitlab.com, bitbucket.org, raw.githubusercontent.com): Allow the content script to run on these code hosting platforms to detect and format .env files.

Children's Privacy

This extension is not directed at children under 13 and does not knowingly collect any information from children.

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top of this page.
