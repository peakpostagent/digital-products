# DotEnv Preview -- Chrome Web Store Listing

## Extension Name
DotEnv Preview

## Short Description (132 chars max)
Syntax-highlight .env files on GitHub. Groups variables by prefix, flags secrets, and adds copy buttons. 100% local.

## Detailed Description

DotEnv Preview automatically formats .env files when you view them on GitHub, GitLab, or Bitbucket -- turning raw key-value pairs into a readable, organized, and security-aware view.

**What it does:**

- Syntax highlighting: Keys, values, comments, and data types each get their own color
- Smart grouping: Variables are automatically grouped by prefix (DB_, API_, AWS_, SMTP_, etc.)
- Sensitive value detection: Real API keys, tokens, and passwords are flagged and masked by default
- Placeholder detection: Values like "changeme", "your_xxx_here", or empty strings are highlighted as needing attention
- Copy buttons: One-click copy for individual values or the entire file
- Search/filter: Quickly find any variable by name or value
- Line numbers: Easy reference for code reviews
- Dark mode support: Automatically matches your GitHub theme
- Toggle view: Switch between formatted and raw view at any time

**Supported file types:**

.env, .env.local, .env.example, .env.production, .env.development, .env.staging, .env.test, .env.sample, .env.template, .flaskenv

**Supported platforms:**

- GitHub (blob view and raw view)
- GitLab
- Bitbucket

**Privacy first:**

All processing happens 100% locally in your browser. No data is collected, transmitted, or stored on external servers. No analytics, no tracking, no network requests.

**Settings:**

- Toggle auto-formatting on/off
- Toggle sensitive value masking
- Toggle prefix grouping
- Toggle line numbers
- Choose light, dark, or auto theme
