Z-Index Inspector - Privacy Policy

Last updated: April 6, 2026

Overview

Z-Index Inspector is a Chrome extension that helps developers visualize z-index values, stacking contexts, and layer conflicts on any web page. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

Z-Index Inspector does not collect, store, transmit, or share any data externally. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No page content or CSS data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.

How It Works

Z-Index Inspector uses the Scripting API to inject a content script into the active tab when you click the extension icon. The script scans the DOM for elements with explicit z-index values, identifies stacking contexts, and renders overlay badges and a sidebar panel. All scanning, analysis, and rendering happens entirely within your browser. User preferences (such as overlay toggle state) are stored locally using chrome.storage.local on your device.

Permissions

The extension requests the following permissions:

  • activeTab — Required to access the current tab when the user clicks the extension icon, allowing the content script to scan and display z-index information.
  • scripting — Used to inject the inspection script into the active tab on demand, rather than running on every page.
  • storage — Used for storing user preferences (such as overlay on/off state) locally on your device.

Third-Party Services

Z-Index Inspector does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero external data collection.

Contact

If you have questions about this privacy policy, you can reach out through the extension's Chrome Web Store listing.
