Service Worker Inspector - Privacy Policy

Last updated: March 28, 2026

Overview

Service Worker Inspector is a Chrome extension that helps developers inspect and debug service worker registrations and cached resources on any web page. Your privacy is important, and this extension is designed to keep all data local to your browser.

Data Collection

Service Worker Inspector does not collect, store, transmit, or share any data externally. Specifically:

  • No personal information is collected.
  • No browsing history is recorded.
  • No service worker or cache data leaves your browser.
  • No analytics or tracking scripts are used.
  • No cookies are set by the extension.
  • No data is sent to external servers.

How It Works

Service Worker Inspector uses the chrome.scripting API to execute scripts in the current tab's page context. These scripts query navigator.serviceWorker.getRegistrations() and the Cache Storage API to gather information about service workers and cached resources. The offline simulation feature temporarily overrides fetch and XMLHttpRequest in the page context. All data is processed locally and displayed in the extension popup. Only temporary UI state (such as offline simulation toggle) is stored in chrome.storage.local and is automatically cleaned up when tabs close.

Permissions

The extension requests the following permissions:

  • storage — Used for temporarily storing UI state (offline simulation toggle) locally on your device.
  • activeTab — Required for the popup to identify the currently active tab and inspect its service workers.
  • scripting — Used to execute inspection scripts in the page context to query service worker registrations and cache storage.

Third-Party Services

Service Worker Inspector does not use any third-party services, APIs, or SDKs.

Changes to This Policy

If this privacy policy is updated, the changes will be posted here with a revised date. The extension will continue to operate on the principle of zero external data collection.

Contact

If you have questions about this privacy policy, you can reach out through the extension's Chrome Web Store listing.
