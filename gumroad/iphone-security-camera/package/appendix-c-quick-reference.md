# Appendix C: Quick-Reference Setup Checklist

Print this page and check off each step as you go. Complete setup takes about 15 minutes.

---

## Phase 1: Prepare the phone (5 minutes)

- [ ] Back up any personal data you want to keep
- [ ] Factory reset: Settings > General > Transfer or Reset iPhone > Erase All Content and Settings
- [ ] Set up the phone with Wi-Fi (skip Apple ID, Face ID, and Siri)
- [ ] Update iOS: Settings > General > Software Update
- [ ] Delete all unnecessary apps

## Phase 2: Optimize settings (3 minutes)

- [ ] Brightness: Set to minimum (Settings > Display & Brightness)
- [ ] Auto-Lock: Set to Never
- [ ] Notifications: Disable all except camera app
- [ ] Do Not Disturb: Enable (Settings > Focus > Do Not Disturb)
- [ ] Background App Refresh: Off for everything except camera app
- [ ] Cellular data: Turn off (Settings > Cellular)
- [ ] Bluetooth: Turn off
- [ ] Siri: Turn off (Settings > Siri & Search)
- [ ] Auto-updates: Turn off (Settings > General > Software Update > Automatic Updates)
- [ ] Reduce Motion: Turn on (Settings > Accessibility > Motion)
- [ ] AirDrop: Turn off

## Phase 3: Install and set up the camera app (5 minutes)

- [ ] Download your chosen camera app from the App Store
- [ ] Open the app and create an account / sign in
- [ ] Set the phone's role to "Camera"
- [ ] Grant permissions: Camera, Microphone, Notifications
- [ ] On your primary phone: install the viewer app and sign in with the same account
- [ ] Verify the live feed appears on your primary phone

## Phase 4: Configure the camera (2 minutes)

- [ ] Select back camera (better quality)
- [ ] Set resolution: 720p (recommended starting point)
- [ ] Enable motion detection: Medium sensitivity
- [ ] Enable push notifications for motion events
- [ ] Test: Walk in front of camera and verify alert on primary phone

## Phase 5: Place and power the camera

- [ ] Choose location (front door, hallway, or other priority area)
- [ ] Check Wi-Fi signal at location (2+ bars minimum)
- [ ] Mount or position the phone (stand, mount, or lean against surface)
- [ ] Plug in charging cable and adapter
- [ ] Plug adapter into surge protector
- [ ] Verify camera angle on primary phone
- [ ] Test motion detection from the final position

## Phase 6: Final verification

- [ ] Live feed visible on primary phone: YES / NO
- [ ] Motion alerts arriving: YES / NO
- [ ] Audio working (if applicable): YES / NO
- [ ] Camera angle covers desired area: YES / NO
- [ ] Phone is charging / plugged in: YES / NO
- [ ] Wi-Fi connection stable: YES / NO

## Weekly maintenance (set a recurring reminder)

- [ ] Restart the camera phone
- [ ] Check battery health (Settings > Battery > Battery Health)
- [ ] Check storage (Settings > General > iPhone Storage)
- [ ] Check for app updates
- [ ] Clean the camera lens
- [ ] Verify the live feed and test motion detection
