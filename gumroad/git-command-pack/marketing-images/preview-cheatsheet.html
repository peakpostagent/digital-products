PeakPost
v1.0

Setup

git config --global user.nameSet identity
git config --global pull.rebase trueRebase on pull

Start repo

git initNew repo
git clone <url>Clone remote

Daily

git status -sShort status
git add -pStage by chunks
git commit -m "..."Commit

Branch

git switch -c feat/xCreate + switch
git switch mainSwitch
git branch -d nameDelete

Remote

git pull --rebaseGet + replay
git push -u originPush + set upstream
git push --force-with-leaseSAFE force

Merge

git merge --squashSquash merge
git merge --abortBail out

Rebase

git rebase mainReplay on main
git rebase -i HEAD~5Interactive

Log

git log --oneline --graphTree view
git log -S "text"Find by content

Undo

git restore file.jsDiscard changes
git reset --soft HEAD~1Undo commit
git revert <hash>Safe undo
