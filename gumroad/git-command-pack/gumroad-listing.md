**4. Drop-In .gitconfig Template** (Text)
50+ productivity aliases, ready to copy. A few favorites:
- `git lg` — beautiful one-line graph log
- `git s` — short status
- `git ap` — interactive staging
- `git pf` — safe force push (`--force-with-lease`)
- `git undo` — undo last commit, keep staged
- `git oops` — pretty reflog for recovering from panic
- `git recap` — what you committed today
- `git cleanup` — delete merged branches
- `git bigfiles` — find the biggest files in your history

Plus modern sensible defaults: `pull.rebase`, `rerere.enabled`, `zdiff3` conflict style, auto-stash on rebase, histogram diff algorithm, color output.

**5. Global .gitignore** (Text)
Covers every OS, editor, and language you're likely to touch. Drop it in once, and you'll never accidentally commit `.DS_Store`, `node_modules/`, `.env`, `.idea/`, `__pycache__/`, Vim swap files, or VS Code settings again — in any repo on your machine, ever.

**6. 5-Minute Quick Start Guide** (Markdown)
Step-by-step setup for macOS, Linux, and Windows. Install the aliases, set your identity, install the global gitignore, try the most useful aliases. You'll be productive in 5 minutes.

**7. README** (Markdown)
Index of what's in the pack and a 60-second orientation.

### Frequently Asked Questions

**Q: What format are the files?**
A: PDF for the cheat sheet. Markdown for the long-form reference, workflow playbook, quick start, and README. Plain text for the `.gitconfig` and `.gitignore` templates. Everything opens in any editor or browser — no special software required.

**Q: Does this work on Windows?**
A: Yes. The Quick Start has separate install steps for macOS, Linux, and Windows (PowerShell). All aliases work in Git for Windows / WSL / PowerShell / Git Bash.

**Q: Which Git version does this target?**
A: Works with Git 2.23+ (the version where `git switch` and `git restore` shipped). If you've installed Git in the last 4 years, you're good. Older versions still work for most commands — only the `switch`/`restore` aliases need substituting with `checkout`.

**Q: Will the aliases conflict with my existing .gitconfig?**
A: The Quick Start shows you how to back up your current config first and either replace it or merge the `[alias]` section. Takes 30 seconds.

**Q: Does this cover GitHub / GitLab / Bitbucket specifics?**
A: The commands are all native git, so they work with any remote. The workflow playbook calls out GitHub's branch protection and squash-merge features specifically.

**Q: How is this different from a free cheat sheet online?**
A: Free cheat sheets give you a list. This pack gives you the cheat sheet PLUS the full reference PLUS the aliases that turn those commands into 2-character shortcuts PLUS the workflow playbook PLUS the global gitignore PLUS the quick-start guide — all cross-referenced, all consistent, all designed to work together.

**Q: Can I customize everything?**
A: Yes. It's all plain text and markdown. Edit as you please. It's your pack.

**Q: Will this get updated?**
A: Yes. Gumroad emails all past buyers when a new version ships. Free updates forever.
