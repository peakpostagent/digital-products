# Marketing Assets - Image Specifications for ComfyUI

## 1. Product Cover Image (1280x720)

**Filename:** `cover-image.png`
**Dimensions:** 1280 x 720 pixels
**Used on:** Gumroad product page header, social media shares

**Design Description:**
- Dark background (deep navy or charcoal, like a code editor theme)
- Title text: "The Developer's AI Prompt Toolkit" in a clean, modern sans-serif font (white or light gray)
- Subtitle: "222 Battle-Tested Prompts" in a smaller accent color (cyan or green, like terminal text)
- Visual element: A stylized terminal or code editor window showing a prompt with `{{PLACEHOLDER}}` markers highlighted in a different color
- Bottom strip: Small logos or text for "ChatGPT | Claude | Ollama" showing compatibility
- Overall aesthetic: Clean, technical, professional. Think VS Code dark theme meets a professional product mockup
- No stock photos of people. Developer tools, not corporate marketing

**Color Palette:**
- Background: #1a1b26 (dark navy) or #0d1117 (GitHub dark)
- Primary text: #e6e6e6 (light gray)
- Accent: #7aa2f7 (soft blue) or #9ece6a (soft green)
- Highlight: #ff9e64 (orange) for placeholder markers
- Secondary: #565f89 (muted blue-gray) for supporting text

## General Notes for ComfyUI Generation

### Style References
- VS Code Dark+ theme color palette
- GitHub dark mode aesthetics
- Vercel and Linear product page design language
- Clean, flat design with subtle gradients and shadows
- No 3D rendered objects or photorealistic scenes
- Typography-focused with code/terminal elements

### Typography
- Headlines: Inter, SF Pro, or similar geometric sans-serif
- Code elements: JetBrains Mono, Fira Code, or similar monospace
- Body text: Clean sans-serif for readability

### Do Not Include
- Stock photos of people
- Overly colorful or busy backgrounds
- AI-generated faces
- Corporate clip art
- Gradients that look like early 2000s web design

### File Formats
- Final output: PNG at 2x resolution (2560x1440 for 1280x720 images)
- Also export at 1x for web use
- Keep file sizes under 500KB per image for fast loading on Gumroad
