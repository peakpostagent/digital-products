============================================
  The Freelancer's Business-in-a-Box
  Setup Instructions
============================================

Thank you for your purchase! Here's how to get started:

1. OPEN THE SPREADSHEET
   - Upload "Freelancer-Business-in-a-Box.xlsx" to Google Drive
   - Right-click the file > Open with > Google Sheets
   - Or open directly in Microsoft Excel or LibreOffice Calc

2. MAKE IT YOURS
   - The spreadsheet contains sample data so you can see how it works
   - Delete the sample rows and replace with your own information
   - Start with the Client CRM tab, then Project Pipeline, then Invoice Tracker

3. READ THE QUICK START GUIDE
   - Open "quick-start-guide.md" for tab-by-tab setup instructions
   - It explains every formula so you understand how things connect

4. USE THE BONUS CHECKLISTS
   - "tax-prep-checklist.md" -- Keep your finances organized for tax season
   - "client-onboarding-checklist.md" -- Start every new client relationship right

WHAT'S IN THIS PACKAGE:
   - Freelancer-Business-in-a-Box.xlsx (7-tab workbook)
   - quick-start-guide.md
   - tax-prep-checklist.md
   - client-onboarding-checklist.md
   - README.txt (this file)

TIPS:
   - The .md files can be opened in any text editor or markdown viewer
   - Yellow cells are for your input; gray cells contain formulas
   - The Dashboard tab updates automatically from all other tabs
   - Dropdown menus are built into Status, Source, Priority, and Category columns

SUPPORT:

Enjoy running your business like a pro!
