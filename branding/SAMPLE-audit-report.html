Overall Health

This report audits Northwind Coffee Co. (SAMPLE) across seven areas that affect security, search ranking, speed, and accessibility. Each area is graded A+ to F. The prioritized action list below tells you exactly what to fix first — and why it matters.

Priority Fixes

#PriorityAreaIssue & recommendation
1 Critical Security Headers Content-Security-Policy: missing
Add a policy starting with default-src 'self' to block injection attacks.
2 Critical Security Headers X-Frame-Options: missing
Add DENY or SAMEORIGIN to prevent clickjacking of your booking page.
3 Critical Structured Data No LocalBusiness structured data
Add schema.org/CafeOrCoffeeShop markup so Google can show your hours, address, and star rating directly in search — huge for a local business.
4 Critical Privacy & Compliance Trackers present but no consent banner detected
The Facebook Pixel fires before consent — a likely GDPR/PIPEDA issue if you get any EU or Canadian visitors. Add a consent banner.
5 Improve Security Headers Strict-Transport-Security: weak
Present but max-age is under 6 months — raise to 31536000.
6 Improve Security Headers Permissions-Policy: missing
Restrict camera/microphone/geolocation access explicitly.
7 Improve SEO & Meta Tags Description 96 chars
A little short — aim for 120-158 characters to fill the search snippet.
8 Improve SEO & Meta Tags Incomplete Open Graph tags
Add og:image so shared links show your storefront photo, not a blank box.
9 Improve Crawlability (robots + sitemap) Sitemap has only 4 URLs
Your menu and location pages aren't in the sitemap — add them so Google indexes them.
10 Improve Image Accessibility 5 of 14 images missing an alt attribute
Your menu-item photos need alt text — for screen readers and Google Image search (where a lot of "coffee near me" traffic starts).
11 Improve Image Accessibility 2 images use a filename as alt
alt="IMG_4821.jpg" tells a screen-reader user nothing — describe the drink.
12 Improve HTTP & Performance Config Using gzip (Brotli would be better)
Brotli saves another 15-25% — one setting on most modern hosts.
